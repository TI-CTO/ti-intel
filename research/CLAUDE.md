# research — 리서치 산출물

Claude Code를 활용한 기술 조사, 비교 분석, 문서화 결과물.

## Structure
- `sessions/` — 개별 리서치 세션 노트 (YYYY-MM-DD_topic-slug.md)
- `summaries/` — 여러 세션을 종합한 정리 (summary_topic-slug.md)
- `references/` — 외부 문서, API 스펙, 논문 저장

## Rules
- 같은 주제 세션이 3개 이상 쌓이면 `summaries/`에 종합 정리 작성
- 유의미한 발견은 `/obsidian-bridge`로 Obsidian 볼트에 동기화
- 외부 소스를 인용할 때는 URL과 접근 날짜 기록
