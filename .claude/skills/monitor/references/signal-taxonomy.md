# Monitor 스킬 — 신호 분류 체계

## 변화 중요도 기준

### 🔴 긴급 (Critical)
즉시 검토가 필요한 신호:
- 경쟁사 상용화 발표 또는 출시
- 핵심 규제 변경 또는 정부 정책 전환
- 기술 돌파구 (TRL 7+ 달성 발표)
- 특허 분쟁 또는 기술 독점 우려

### 🟡 주목 (Warning)
모니터링 강화가 필요한 신호:
- 주요 기업 파일럿/PoC 발표
- 신규 표준화 제안 또는 컨소시엄 결성
- 인용 급증 논문 발표 (전월 대비 2배+)
- 주요 특허 출원 급증 (30일 내 10건+)
- 시장 조사 기관의 하향/상향 리포트

### 🟢 유지 (Normal)
- 점진적 뉴스 업데이트
- 기존 동향의 연속선상 발전
- 학술 논문의 점진적 축적

## 신뢰도 우선순위
```
1순위: [A] 기업 공식 발표, 정부 정책 문서
2순위: [B] 주요 기술 언론 (TechCrunch, IEEE, MIT Tech Review)
3순위: [C] 분석가 블로그, 산업 뉴스레터
```
