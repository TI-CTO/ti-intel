# Discover 스킬 — 분석 프레임워크 참조

## 우선순위 매트릭스 기준

### 전략 가치 평가 (1~10)
| 점수 | 기준 |
|------|------|
| 9~10 | 핵심 사업과 직결, 경쟁 차별화 필수, 5년 내 시장 10배 이상 |
| 7~8 | 사업 연계 명확, 경쟁사 대비 우위 확보 가능 |
| 5~6 | 잠재 기회, 방향성은 맞으나 확신 불충분 |
| 3~4 | 주변부 기회, 핵심 역량과 연계 불분명 |
| 1~2 | 트렌드 편승 수준, 전략적 가치 낮음 |

### 실행 가능성 평가 (1~10)
| 점수 | 기준 |
|------|------|
| 9~10 | TRL 7~9, 내부 역량 보유, 즉시 착수 가능 |
| 7~8 | TRL 5~6, 소규모 역량 보완으로 착수 가능 |
| 5~6 | TRL 3~4, 파트너십/M&A 필요 |
| 3~4 | TRL 1~2, 기초 연구 단계 |
| 1~2 | 기술 실현 가능성 불명확 |

## 3B 전략 판단 기준
```
Build if: 차별화 중요도 > 8/10 AND 내부역량 보유 AND 시장여유 > 18개월
Buy if:   시장진입 긴급도 > 8/10 OR 기술격차 > 2년
Borrow if: 위 둘 다 해당 없을 때 (파트너십, 오픈소스 활용)
```

## TRL 4사분면
```
         High TRL (7~9) — 상용화 가능
              │
   [유지]     │     [베팅]
   방어적 투자 │  공격적 투자
              │
──────────────┼──────────────
              │
   [탐색]     │     [Watch]
   R&D 관망   │  기술 추적
              │
         Low TRL (1~6) — 연구/개발 단계
   Low Disruption ←──→ High Disruption
```

## 신뢰도 태그
```
[A] Official   — 기업 공시, 정부 통계, peer-reviewed 논문
[B] Reputable  — 주요 언론, 애널리스트 리포트
[C] Indicative — 블로그, 미검증 보도자료
[D] Unverified — 단일 소스, 루머
```
