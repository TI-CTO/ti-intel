# Secure AI 위닝테크 과제 제안서

## 1. 과제 기본 정보

| 항목 | 내용 |
|------|------|
| 과제명 | Secure AI |
| 기술 도메인 | Security |
| 타겟 고객 | LG U+ 통화 서비스 이용 고객 (B2C) |
| 목표 시기 | 미정 |

## 2. 고객가치

### 핵심 니즈 (Pain Point)
- 통화 중 개인정보 보안 위협
- 보이스피싱 등 금전적 피해 차단 필요

### 제공 가치
- 양자암호화 + 동형암호 기술로 고객 개인정보를 안전하게 보호
- OndeviceAI 기반 탐지/차단으로 실시간 안심 통화 제공

## 3. 핵심 기술

| 기술 | 역할 |
|------|------|
| 양자암호화 | 통화 데이터 암호화 |
| 동형암호 | 암호화 상태에서 데이터 처리 |
| OndeviceAI | 단말 단에서 실시간 보이스피싱 탐지/차단 |

## 4. 사업 포텐셜

### 시장 기회
- 국내 보이스피싱 피해액 급증 추세
- 통화 보안 서비스 수요 증가

### 사업 목표
- 고객 필수 서비스로 자리매김

## 5. 기술 경쟁력

### 알고 있는 경쟁사
- 국내: SKT, KT
- 해외: 에이전트 판단에 위임

### 차별화 포인트
- 양자암호 + 동형암호 + OndeviceAI 3중 보안 조합
